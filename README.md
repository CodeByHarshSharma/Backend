# Backend Development Journey (Day 1 - Day 7)

This repository contains my first seven days of learning backend development from scratch using **Node.js**, **Express.js**, and **MongoDB**. Each day's folder covers small concepts and builds on the previous one through hands-on practice.

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Postman (API Testing)

## 📚 What I Learned

* Setting up a Node.js server
* Express.js routing and middleware
* Building REST APIs
* Connecting MongoDB
* Performing CRUD operations
* Testing APIs with Postman
* Basic backend project structure

## 📝 Final Implementation

On **Day 7**, I built a minimal **Notes App** implementing complete CRUD functionality with MongoDB and tested all API endpoints using Postman.

This repository marks the beginning of my backend development journey and serves as a record of my learning and progress.
